Explanation:
The event triggers are the strings that are not part of the text.

## Solution

```python
import re

def extract_event_triggers(text):
    event_triggers = []
    for match in re.finditer(r'[^\w\s]+', text):
        event_triggers.append(match.group())
    return event_triggers
```
